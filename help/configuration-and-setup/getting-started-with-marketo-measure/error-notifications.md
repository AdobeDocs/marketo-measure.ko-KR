# 오류 알림 {#error-notifications}

다음은 인앱 알림 또는 이메일을 통해 받을 수 있는 오류 목록입니다. 이러한 메시지가 표시되면 각각의 문제 해결 단계를 따르십시오. 이러한 단계로 문제가 해결되지 않으면 다음 연락처에 문의하십시오. [Marketo 지원](https://nation.marketo.com/t5/support/ct-p/Support).
