   구매자 속성 터치포인트 하위 그리드 - 계정, 기회 및 연락처\

   >
   >이 설정을 수정할 때는 작은 일괄 처리 크기가 CRM에서 더 많은 API 크레딧을 사용한다는 점에 유의하십시오. CRM에서 CPU 시간 초과 또는 높은 CPU 로드가 발생하는 경우에만 일괄 처리 크기를 줄이는 것이 좋습니다.
